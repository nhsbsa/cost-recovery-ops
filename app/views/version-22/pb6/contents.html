PB6 - Updating Person Details

Person level

  • Journey 1 - person has 0 treatments attached to any claims
  • Journey 2 - person has 1 or more treatments attached to 1 claim
  • Journey 3 - person has multiple treatments attached to multiple claims - should we still be able to edit address from here?

What will be updated

  • Everything

What wont be updated

  • Nothing

Claim level

User research findings:

  • All person details will be correct before adding treatments to a claim
  • Once a claim is sent if any changes need to be made this will be done via resub level

Based on the the above this journey may no longer be needed

Resubmission level

  • Journey 1 - via reasons for contestations screen (before reasons added)
  • Journey 2 - via reasons for contestations screen (after reasons added)
  • Journey 3 - via treatment details screen

What will be updated

  • Resubmission currently being worked on
  • All treatments for that person which are attached to the resubmission currently being worked on
  • The claim the resubmission is attached to
  • Resub case history

What wont be updated

  • Central person details
  • Person details case history
  • Any completed claims
  • Any treatments attached to any completed claims
  • Any pending claims
  • Any treatments attached to any pending claims
  • Any other resubmissions with treatments attached for that person
  • Any treatments for that person not currently attached to a claim
  • All entitements
